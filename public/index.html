OpenAI Realtime Demo

This is a realtime demo of voice-powered function calling using Cloudflare Workers, Replicate, and the OpenAI Realtime API

Try these voice commands:

"Change the background to the color of the sky"
"Change the text to the color of a polar bear"
"Create an image of a sloth wearing sunglasses"
